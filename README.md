## 📌 Descrição
Este projeto tem como objetivo auxiliar produtores rurais no controle de perdas na colheita de cana-de-açúcar.

A solução simula uma aplicação de Agrotech, ajudando na tomada de decisão com base em dados.

---

## 🚜 Problema
As perdas na colheita de cana podem chegar a até 15%, impactando diretamente o lucro do produtor.

---

## 💡 Solução
Sistema em Python que:
- Registra colheitas
- Calcula perdas automaticamente
- Armazena dados em JSON
- Gera análises de eficiência

---

## 🧠 Conceitos aplicados

- Funções (subalgoritmos)
- Estruturas de dados (listas e dicionários)
- Manipulação de arquivos (JSON)
- Validação de dados
- Interface via terminal

---

## 🗄️ Persistência de Dados

O sistema utiliza arquivos JSON como forma de armazenamento, simulando um banco de dados.

Essa abordagem foi adotada pela simplicidade e facilidade de uso em ambiente local, mantendo os conceitos de persistência exigidos na atividade.
